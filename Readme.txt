Projekt je možné vyskúšať na webstránke https://juraj-juraj.github.io/fyo_projekt/
Avšak je možné ho spustiť aj lokálne pomocou http servera. 
Môžete použiť hocijaký program na spúšťanie http servera, napríklad XAMPP alebo Python.
Ukážka ako to spustiť pomocou pythonu.
1. Nainštalujte si Python (ak ho ešte nemáte nainštalovaný).
2. V tejto zložke v konzoli spustite príkaz:
   py -m http.server 8081
3. V prehliadači otvorte adresu:
   http://0.0.0.0:8081
